# FolderBombCreator
FolderBombCreator is a tool for creating a customizable number of folders at any location with a single click.
When you finish, press <b>Load</b>, and your folders start creating.

<img width="108" height="42" alt="image" src="https://github.com/user-attachments/assets/1dc44cb3-e1b8-4369-bd47-b41dba245370" />
